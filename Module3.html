  1. SHANNON FANO CODING:

    Procedure

    1.The messages are first written in the order of decreasing probability 2.Then divide the messages set into two most equiprobable subset X and Y. 3.The message of 1st set X is given bit 0 and message in the 2nd subset is given bit 0. 4.The procedure is now applied for each set seperatly till end. 5.Finally we get the code word for respective symbol. 6.Then calculate Entropy,Efficiency,Redundancy,Code Variance.

  2. Huffman Code:

    Procedure

    With the Huffman code in the binary case the two least probable source output sysmbols are joined together,resulting in a new message alphabet with one less symbol

    1.Take together smallest probabilities : P(i)+ P(j)

    2.Replace symbol i and j by new symbol

    3.Go to 1 until the end.

    4.Generate codeword and length for respective symbol

    5.Calculate Average lenth of the codeword

    6.Calculate Efficiency,Redundancy and Code Variance
